Elementos del Sitio Web

Analogía: La Anatomía de una Casa

Un sitio web se divide típicamente en tres grandes bloques, igual que una casa:

  • Header (Techo): Visible desde lejos, tiene el nombre de la casa (Logo) y te protege.
  • Main (Habitaciones): Donde vives, comes y duermes. El contenido real.
  • Footer (Cimientos): Sostiene todo, tiene la info técnica (tuberías/copyright) y contactos.
Explicación Técnica Selecciona una parte para ver su función específica. Contiene el Logo y la Navegación Principal. Se repite en todas las páginas. Contiene el contenido único de la página actual. Cambia según la ruta. Cierra la página. Enlaces legales, mapa del sitio y redes sociales.
